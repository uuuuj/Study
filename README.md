# Study

**실무에서 겪었던 문제 해결 과정, 이슈 트래킹, 개념 정리 등을 기록하는 개인 기술 기록 저장소**

이 저장소는 실무에서 마주친 문제들(보안 관련 민감 내용 제외)을 정리하고, 그 과정에서 학습한 개념과 해결 방법을 기록하기 위한 **기술 노트/로그 저장소**입니다.

대부분의 기록은 AI 기반으로 작성되지만, 반드시 제가 검토한 후 커밋합니다.

---

## 저장소 목적

이 저장소는 다음과 같은 목적을 위해 사용됩니다:

- 실무에서 경험한 기술 문제와 해결 과정 기록
- 해당 문제의 원인 분석 및 해결 방법 정리
- 관련 개념/이론 요약
- 재사용 가능한 코드/스크립트 및 참고 자료 정리

업무 중 마주친 문제라면 **보안적으로 민감한 내용**은 제외하고 기록합니다.

---

## 기록 예시

실전에서 있었던 다양한 이슈를 다음과 같은 형태로 기록합니다:

- 특정 서비스가 재기동 없이 응답하지 않던 현상 원인 분석
- Django ORM이 특정 쿼리를 비효율적으로 생성하는 문제 해결
- 리눅스 시스템 콜/권한 문제로 인한 파일 접근 오류 해결
- 배포 파이프라인 에러 로그 분석과 조치

> *보안에 민감한 정보(비밀번호, 토큰, 고객 데이터 등)는 제거/각색하여 기록합니다.*

----

기본적으로 "실무에서 마주친 문제와 그 해결 과정"을 중심으로 기록하는 저장소입니다.
